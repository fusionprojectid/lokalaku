# Security Policy

## Supported Scope

This repository contains a static marketing and booking-support website for LokaLaku Trip.

Security maintenance covers:

- `index.html`
- `privacy-policy.html`
- `terms-conditions.html`
- `booking-help.html`
- `thank-you.html`
- CSS and JavaScript files in `css/` and `js/`
- Local visual assets and icon/font assets used by the website

## Supported Versions

| Version | Supported |
| --- | --- |
| Current production website | Yes |
| Older local copies or forks | No |

## Reporting a Vulnerability

If you find a security issue, please report it responsibly.

Preferred contact:

- WhatsApp: [+62 813-3142-1031](https://wa.me/6281331421031)
- Instagram: [@lokalaku_trip](https://www.instagram.com/lokalaku_trip/)
- Website contact form: [https://lokalaku.vercel.app/#contact](https://lokalaku.vercel.app/#contact)

Please do not publicly disclose the issue until we have had time to review and address it.

## What To Include

Please include:

- A short description of the issue
- The affected page or file
- Steps to reproduce
- Potential impact
- Screenshots, request examples, or logs if useful
- Suggested fix, if you already have one

## Response Timeline

We aim to:

- Acknowledge reports within 72 hours
- Confirm reproducible issues as soon as possible
- Patch verified website issues within 7 to 14 business days when practical

Complex issues involving third-party providers may take longer because they can depend on external services.

## Third-Party Services

The website may use or link to third-party services, including:

- Formspree for forms
- WhatsApp for booking conversations
- Instagram and social media links
- YouTube embeds
- Google AdSense
- Vercel hosting and analytics tools
- Font Awesome CDN on legal/help pages

If the vulnerability is inside a third-party service, please also report it to that provider through their official security channel.

## Responsible Disclosure

Do not:

- Access, modify, or delete data that does not belong to you
- Run destructive tests
- Spam forms or WhatsApp links
- Attempt social engineering against staff, customers, or partners
- Publicly disclose an unfixed vulnerability without coordination

Thank you for helping keep LokaLaku Trip and its visitors safe.
